# .decapod - Decapod Project Metadata

This directory is managed by the Decapod CLI.

## Contents

- `constitutions/`: Overrides for the embedded Decapod methodology.
  - You can edit these files to customize how Decapod behaves for this project.
- `data/`: **DO NOT TOUCH**. This is persistent storage (SQLite databases and event logs) for the Decapod CLI.
- `generated/`: Artifacts generated by Decapod.
